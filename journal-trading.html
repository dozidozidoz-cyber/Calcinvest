Persistance : si vous êtes connecté avec un compte CalcInvest, vos trades sont synchronisés via Supabase (chiffrement TLS, RLS row-level activé — chaque user ne voit que ses trades). Sans compte, le journal vit en localStorage du navigateur (limité, pas de sync inter-appareils).
Calculs : Expectancy = (winrate × avg_win) − (lossrate × |avg_loss|). Profit Factor = gains totaux / pertes totales. Sharpe annualisé approximé en supposant 252 jours de trading/an.
Schema Supabase : à exécuter dans le SQL Editor du projet → scripts/supabase_migrations/001_trades.sql
